Class Widget

The base class of the Phosphor widget hierarchy.

A widget wraps an absolutely positioned DOM node. It can act as a container for child widgets which can be arranged with a Phosphor layout manager, or it can act as a leaf control which manipulates its DOM node directly.

A root widget (a widget with no parent) can be mounted anywhere in the DOM by calling its attach method and passing the DOM node which should be used as the parent of the widget's node.

    • Defined in src/widgets/widget.ts:560

    Calculate the preferred maximum size for the widget.

    This is used by Phosphor's layout machinery to compute the maximum space allowed for the widget and its children. This is independent of and subordinate to the maximum size specified in CSS. User code will not typically interact with this method.
